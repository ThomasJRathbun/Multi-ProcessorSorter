\section{NAME}
\begin{center}
sorter - sorts all .csv files inside of a directory tree  
\end{center}

\section{SYNOPSIS}
\begin{center} 
./sorter [OPTION]... 
\end{center}

\section{DESCRIPTION}
\begin{center} 
The sorter program traverses the directory tree starting at current
directory unless -d is specified.

OPTIONS
\begin{itemize}
\item -c
\begin{itemize}
\item header to sort by
\end{itemize}
\item -d
\begin{itemize}
\item directory to sort
\end{itemize}
\item -o
\begin{itemize}
\end{itemize}
\item the output directory for all sorted files
\end{itemize}

\end{center}
